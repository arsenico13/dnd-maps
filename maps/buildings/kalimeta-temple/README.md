### ENG

"Kalimeta Temple" is a `35x20` squares map.

In this repo there's the preview of the map.
You can get this map and all the variants at this link: [Kalimeta Temple DriveThruRPG](https://www.drivethrurpg.com/product/370194/Kalimeta-Temple--Arsenico13-Free-Battlemap--September-2021)
There you can find:
- day/night version
- grid/gridless version
- MapTool maps ready to use with VBL and lights


All maps in this bundle are exported at a scale of 100 pixels per square (3500x2000 pixels).


__If need to use this with a VTT with squares at 70 pixels (for 5′ squares), resize it to 2450x1400 pixels.__

### ITA

"Kalimeta Temple" è una mappa con griglia da `35x20` celle.

In questo repo trovi la preview della mappa.
Puoi trovare la mappa in tutte le sue versioni al seguente link: [Kalimeta Temple DriveThruRPG](https://www.drivethrurpg.com/product/370194/Kalimeta-Temple--Arsenico13-Free-Battlemap--September-2021)
Lì puoi trovare:
- versione giorno/notte
- versione con o senza griglia
- mappa per MapTool già pronta per essere importata, con VBL e luci


Tutte le mappe di questo bundle sono esportate ad una scala di 100 pixel per cella (per un totale di 3500x2000 pixel).


__Per usare una di queste mappe in un VTT con celle da 70 pixel l'una, ridimensiona l'immagine a 2450x1400 pixel.__
